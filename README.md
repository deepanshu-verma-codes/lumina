# 💎 Lumina

**Lumina** is a premium, Loom-inspired Chrome extension designed for high-quality screen recording and seamless screenshot capture. Built with a focus on performance and a modern user experience, Lumina provides a suite of professional tools directly in your browser.

---

## ✨ Features

- 🎥 **Screen Recording:** Capture your entire screen or specific tabs with ease.
- 📸 **Smart Screenshots:** One-click capture of your current active tab.
- 🖼️ **Webcam Overlay:** Professional, draggable circular webcam bubble for personalized presentations.
- ⏱️ **Live Timer:** Track your recording duration in real-time within the popup and on-page controls.
- ⏸️ **Pause & Resume:** Full control over your recording sessions.
- 🖱️ **Floating Controls:** On-page control panel to manage recordings without switching tabs.
- 📜 **Activity History:** Keep track of your last 10 captures with the built-in history log.
- ⌨️ **Keyboard Shortcuts:**
  - `Alt + Shift + S`: Take a Screenshot
  - `Alt + Shift + R`: Toggle Recording
- 🎨 **Modern UI/UX:** A sleek, dark-themed interface featuring a new **Aperture logo** and interactive animations.

---

## 🛠️ Tech Stack

Lumina is built using pure web technologies to remain lightweight and fast:
- **Vanilla JavaScript (ES6+):** Modular architecture with specialized services.
- **HTML5 & CSS3:** Modern layout using Flexbox, Grid, and CSS Variables.
- **Chrome Extension API:** Built on **Manifest V3** for maximum security and longevity.
- **SVG Icons:** High-quality, scalable vector graphics for all UI elements.

## 🚀 Installation

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/deepanshu-verma-codes/lumina.git
   ```
2. **Open Chrome Extensions:**
   Navigate to `chrome://extensions/` in your browser.
3. **Enable Developer Mode:**
   Toggle the switch in the top right corner.
4. **Load the Extension:**
   Click **Load unpacked** and select the root folder of the project.

---

## 📖 How to Use

1. Click the **Lumina** icon in your extension bar to open the popup.
2. Choose your mode (Screen or Camera).
3. Toggle **Microphone** or **Webcam Overlay** as needed.
4. Click **Start Recording** or use the shortcut `Alt + Shift + R`.
5. Use the **Floating Controls** on the page to pause or stop.
6. Once stopped, your recording will automatically download.
